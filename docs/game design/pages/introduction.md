# Introduction
Towersmith is a tower defense game, with heavy focus on tower customization and replayabilty.

The goal of the game is to fend off waves of enemy attackers. If the enemies manage to destroy the player's castle, the game is lost.

In order to stop these attackers, the player can place towers that will shoot at them, gold mines that provide additonal resources, and more. All of these buildings can be upgraded and customized to fit the player's grand design. Once all enemies are defeated, the level is complete.
